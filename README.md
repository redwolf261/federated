# FLEX-Persona

FLEX-Persona is a federated learning research project for heterogeneous client architectures and non-IID data.

## Current Status

Stage 1 (Waterfall): Project foundation and configuration scaffolding.

## Planned Package Layout

- `flex_persona/config` for experiment configuration schemas.
- `flex_persona/utils` for reusable helpers (seed, constants, shared typing).

## Next Stages

1. Data layer for FEMNIST and CIFAR-100 client partitions.
2. Model layer with heterogeneous backbones and adapters.
3. Prototype, similarity, clustering, federated orchestration, training, and evaluation.
