FLEX-Persona enables federated learning across heterogeneous clients with non-IID data through representation-based collaboration. Instead of sharing model parameters, clients exchange compact representation summaries, enabling seamless cross-architecture collaboration while preserving privacy and improving personalization.

Key Capabilities

◆

Local Client Training

Each client optimizes independently on local data, preserving autonomy and enabling heterogeneous architectures.

◈

Representation Adapter

Neural networks map diverse local features into a shared, low-dimensional latent space for cross-client alignment.

⬢

Secure Representation Sharing

Compact summary distributions replace full parameter sharing, dramatically reducing bandwidth and communication overhead.

◊

Dynamic Clustering

Server automatically groups similar clients using Wasserstein distance on prototype distributions for optimal collaboration.

★

Cluster-Level Aggregation

Shared knowledge derived from peer clusters guides personalized updates while maintaining client-specific specialization.

◆

Personalized Head Fine-Tuning

Individual task-specific layers per client balance global alignment with local optimization for superior personalization.
